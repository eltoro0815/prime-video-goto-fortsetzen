# Amazon Prime Video - Fortsetzen-Kategorie nach oben verschieben

Ein Tampermonkey-Skript, das die "Fortsetzen"-Kategorie auf Amazon Prime Video direkt nach dem Hero-Bereich platziert, damit man nicht scrollen muss, um zu sehen, welche Serien und Filme man fortsetzen kann.

## Installation

1. Installiere [Tampermonkey](https://www.tampermonkey.net/) für deinen Browser
2. Öffne die Datei [prime-video-goto-fortsetzen.user.js](./prime-video-goto-fortsetzen.user.js) im Repository
3. Klicke auf "Raw"
4. Tampermonkey sollte automatisch die Installation anbieten
5. Klicke auf "Installieren"

## Features

- Verschiebt die "Fortsetzen"-Kategorie direkt unter den Hero-Bereich
- Scrollt automatisch zur richtigen Position
- Funktioniert auch bei dynamischen Seitenänderungen

## Kompatibilität

Getestet mit:
- Chrome und Firefox
- Amazon Prime Video Deutschland
